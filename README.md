# 🍦 Ice Cream Shop Website

A responsive and visually appealing **Ice Cream Shop Website** built using **HTML, CSS, and JavaScript**.
This project showcases an online ice cream shop where users can explore different ice cream flavors, view services, and contact the shop.

---

## 🚀 Features

* 🍨 Attractive landing page
* 🧁 Ice cream products showcase
* 📸 Gallery section
* 🧑‍🍳 Services information
* 📞 Contact page
* 📱 Fully responsive design
* ⚡ Smooth navigation

---

## 🛠️ Tech Stack

* **HTML5** – Website structure
* **CSS3** – Styling and layout
* **JavaScript** – Interactive features
